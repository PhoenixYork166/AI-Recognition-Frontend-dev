### 1. To download this React app:
```bash
git clone https://github.com/PhoenixYork166/AI-Recognition.git;
```

### 2. To install dependencies:
```bash
npm install;
```

### 3. To swap between git branch prod & git branch dev
```bash
git checkout dev;
```
```bash
git checkout prod;
```

### 4. To run this app locally:
```bash
npm start
```
